# ♟️ Sistema de Xadrez em Java

Projeto desenvolvido em Java com foco em **Programação Orientada a Objetos (POO)**, simulando uma partida completa de xadrez no terminal.

Este sistema implementa as principais regras do jogo, movimentação das peças, validações e mecânicas especiais, servindo como excelente projeto prático para estudos de Java.

---

## 📌 Sobre o Projeto

O objetivo deste projeto foi aplicar conceitos essenciais de desenvolvimento backend utilizando Java, como:

- Classes e Objetos
- Encapsulamento
- Herança
- Polimorfismo
- Coleções 
- Tratamento de Exceções
- Enumerações
- Lógica de negócio
- Organização em camadas/pacotes

---

## ♟️ Funcionalidades

✔️ Tabuleiro de xadrez 8x8  
✔️ Movimentação de peças válida  
✔️ Captura de peças  
✔️ Controle de turnos  
✔️ Xeque  
✔️ Xeque-mate  
✔️ Promoção de peão  
✔️ Roque pequeno e grande  
✔️ En Passant  
✔️ Interface via terminal
